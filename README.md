# Romantic Valentine's Day Web App

This project is a romantic web experience for your girlfriend.

## Setup Instructions

1.  **Add Your Assets**:
    Navigate to the `public/assets` folder and add your files. Rename them exactly as follows:
    -   `asset1.jpg` (Internship)
    -   `asset2.jpg` (Delhi)
    -   `asset3.jpg` (3-Hour Calls)
    -   `asset4.jpg` (Bangalore)
    -   `asset5.jpg` (Chocolate Cake)
    -   `asset6.jpg` (Clumsiness)
    -   `asset7.jpg` (Lalbagh)
    -   `asset8.mp4` (Fav Vlogger - **Video**)
    -   `asset9.jpg` (Rohan Iksha Meet)
    -   `music.mp3` (Background Music)

    *Note: If your images are `.png` or other formats, you will need to update `main.js` to match the filenames.*

2.  **Run the Project**:
    Open a terminal in this folder and run:
    ```bash
    npm run dev
    ```
    Then open the link shown (usually `http://localhost:5173`).

3.  **Build for Production**:
    To create a deployable version, run:
    ```bash
    npm run build
    ```
